# SSII - Curso 25/26 - Security Team 5

Este repositorio centraliza los proyectos correspondientes a las actividades de **PAI** (Proyectos de Aseguramiento de la Información) de la asignatura Seguridad en Sistemas Informáticos y en Internet en la Escuela Técnica Superior de Ingeniería Informática de la Universidad de Sevilla, curso 2025-2026.

## Estructura del repositorio

El repositorio está organizado por módulos independientes para cada entrega:

docs/: Documentación técnica, manuales y presentaciones de cada proyecto (ej. PAI-1-Presentación_v1.1.pdf).

python/PAI-X/: Código fuente organizado por proyecto. Cada proyecto puede tener las siguientes subdirectorios:

- data/: Almacenamiento de datasets y archivos de integridad (ej. .hmac).

- tests/: Suite de pruebas unitarias y de integración.
